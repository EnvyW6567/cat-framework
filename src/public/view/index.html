이 사이트는 Cat Framework 로 작성된 웹 어플리케이션 서버에서 서빙됩니다.

Cat Framework는 Low-Level로 구현된 Http 모듈을 사용합니다.

Express, SpringBoot의 특징을 결합한 친숙한 인터페이스를 제공합니다.

독자적인 방식으로 의존성 주입 및 컨테이너를 관리하며 제어 역전을 실현합니다.

Cat Framework(캣 프레임워크)
